# Web_Wonders

# 🦖 Jurassic Website

A full-stack web application based on the Jurassic era, featuring dinosaurs, prehistoric life, paleontology, quizzes, and interactive educational content.

## 📁 Project Structure

```text
jurassic-website/
│
├── frontend/      # React + Vite
├── backend/       # Express.js
├── .gitignore
└── README.md
```

---

## 🛠️ Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS (Planned)

### Backend

- Node.js
- Express.js

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
cd jurassic-website
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

### Backend

```bash
cd backend
npm install
npm run dev
```

Runs on:

```
http://localhost:5000
```
